# Диаграмма состояний


На скриншоте ниже представлена диаграмма состояний для конечного автомата:
![диаграмма состояний](https://github.com/user-attachments/assets/7249b4f3-dbaa-40e4-8e44-f509634ccbd3)

Краткое описание:
1. Normal (начальное состояние) - считываем текущий символ и относительно него переводим автомат в соответствующее состояние.
2. InIdentifier (состояние идентификатора) - накапливаем буквы или цифры.
3. InWhitespace (состояние разделителя) - копим пробелы и формируем соответствующий токен.
4. InError - пропуск символов до конца строки или до появления допустимого символа

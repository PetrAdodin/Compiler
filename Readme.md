## -------- Название и цель лабораторной работы --------

Название: Разработка пользовательского интерфейса (GUI) для языкового процессора

Цель: Создание кроссплатформенного графического интерфейса (GUI) для языкового
процессора в виде специализированного текстового редактора.

## -------- Описание проекта --------

Текстовый редактор с пользовательским интерфейсом (на данном этапе)



## -------- Используемые технологии --------

Программа написана на C# с использованием .NET Framework (WPF) в среде
разработки Visual Studio



## -------- Инструкция по сборке и запуску --------

1. Нужен Visual Studio 2022 или 2019 (можно Community версию)
2. Открыть файл решения Compiler_1.sln
3. Нажать F5 или "Запуск" в меню
Если нет Visual Studio, можно скачать её с официального сайта Microsoft,
она бесплатная для студентов.



## -------- Руководство пользователя --------

1. Запускаете программу
2. Пишете код в верхнем поле (по умолчанию там пример print ('Hello world!'))
3. Нажимаете кнопку с зелёной стрелкой (или Пуск в меню) // WIP
4. Смотрите результат в таблице снизу

Файлы можно открывать через меню Файл -> Открыть или кнопкой с папкой.
Сохранять - через Сохранить или Сохранить как.
Если просто нажать Сохранить для нового файла, откроется диалог сохранения.
